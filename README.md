# 교회 재정 관리

교회 재정 담당자를 위한 웹 기반 수입·지출·예산 관리 프로그램입니다. 일요일 기준 주차별 실적을 입력하고, 연간 예산과 기간 통계를 확인하며, 엑셀 출력과 JSON 백업으로 데이터를 관리할 수 있습니다.

## 기술 스택

- **Frontend**: React 19, TypeScript, Vite
- **Backend**: Express API(로컬), Vercel Serverless API(배포)
- **Storage**: SQLite(로컬), Vercel Blob JSON(배포)
- **Export**: ExcelJS 기반 `.xlsx` 다운로드

## 실행 방법

자세한 설치·실행 안내는 **[docs/설치및실행.md](docs/설치및실행.md)** 를 참고하세요.

Windows에서는 **`실행.bat`** 더블클릭으로 설치, 서버 실행, 브라우저 열기를 진행할 수 있습니다.

```bash
npm install
npm run dev
```

기본 주소는 다음과 같습니다.

| 구분 | 주소 |
|------|------|
| 웹 화면 | `http://localhost:5173` |
| API 서버 | `http://localhost:3001` |

## 로그인

프로그램 접속 시 관리자 비밀번호가 필요합니다.

- 최초 기본 비밀번호: `9999`
- 로그인 세션은 쿠키로 유지됩니다.
- 비밀번호 변경과 로그아웃을 지원합니다.
- 비밀번호 변경은 **백업** 메뉴에서 할 수 있습니다.

## 주요 기능

### 월별 실적

- 선택한 월의 모든 일요일 주차를 한 화면에서 입력합니다.
- 수입과 지출 계정을 분리해 표시합니다.
- 월 합계, 연간 예산, 월 요약을 함께 확인할 수 있습니다.
- 1월은 전년 잉여를 포함하고, 2월 이후는 전월 순잉여 이월을 반영해 요약합니다.
- 지출 계정별 월간 적요를 입력할 수 있습니다.
- 월별 실적을 `.xlsx` 파일로 다운로드할 수 있습니다.

### 연간 예산

- 연도별 계정 예산을 입력합니다.
- 계정별 연누계 실적과 집행률을 확인합니다.
- 수입·지출 계정 구조에 맞춰 예산을 관리합니다.

### 기간 통계

- **월별**: 선택 월의 주차별 실적, 기간합계, 연누계, 연간예산, 집행률
- **상반기**: 1월부터 6월까지 월별 실적과 예산 대비 현황
- **연간**: 1월부터 12월까지 월별 실적과 예산 대비 현황
- 기간 통계를 `.xlsx` 파일로 다운로드할 수 있습니다.

### 계정 관리

- 수입·지출 계정을 대분류와 세부 계정으로 관리합니다.
- 계정 추가, 이름 변경, 삭제를 지원합니다.
- `#대분류`, `-계정명` 형식의 벌크 입력을 지원합니다.
- 계정 구조를 JSON으로 Export/Import할 수 있습니다.

### 백업 및 초기화

- 계정, 주간 실적, 지출 적요, 연간 예산을 JSON 파일로 백업·복원합니다.
- 기본 계정 템플릿으로 초기화할 수 있습니다.
- 관리자 비밀번호를 변경할 수 있습니다.

## 데이터 저장

### 로컬 실행

- 재정 데이터: `data/finance.db`
- 인증 데이터: `data/auth.json`
- 서버를 다시 실행해도 SQLite DB에 데이터가 유지됩니다.
- 다른 PC로 옮길 때는 DB 파일을 함께 복사하거나 **백업** 메뉴의 JSON Export/Import를 사용하세요.

### Vercel 배포

- 재정 데이터와 인증 데이터는 Vercel Blob에 JSON 형태로 저장됩니다.
- Vercel에서 실사용하려면 Blob Storage 연결이 필요합니다.
- 자세한 내용은 **[docs/vercel-배포.md](docs/vercel-배포.md)** 를 참고하세요.

## 빌드 및 미리보기

```bash
npm run build
npm run preview
```

`npm run preview`는 API 서버와 Vite preview를 함께 실행합니다.

## 개발 명령

| 명령 | 설명 |
|------|------|
| `npm run dev` | API 서버와 웹 화면을 함께 실행 |
| `npm run dev:server` | Express API만 실행 |
| `npm run dev:client` | Vite 개발 서버만 실행 |
| `npm run build` | 프론트엔드 빌드 |
| `npm run preview` | 빌드 결과 미리보기 |
| `npm start` | 프로덕션 API 실행 |
| `npm run rebuild:sqlite` | `better-sqlite3` 네이티브 모듈 재빌드 |
| `npm run import:md` | `docs/수입.md`, `docs/지출.md` 계정 데이터를 DB로 가져오기 |
