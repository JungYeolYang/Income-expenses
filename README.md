# 교회 재정 관리 (Income-expenses)

웹 브라우저에서 동작하는 주간 수입·지출 입력 및 연간 예산 관리 프로그램입니다. 데이터는 **SQLite** (`data/finance.db`)에 저장됩니다.

## 실행 방법

자세한 설치·실행 안내는 **[docs/설치및실행.md](docs/설치및실행.md)** 를 참고하세요.

Windows에서는 **`실행.bat`** 더블클릭으로 서버 실행 + 브라우저 열기가 가능합니다.

```bash
npm install
npm run dev
```

브라우저에서 표시된 주소(기본 `http://localhost:5173`)로 접속합니다.

## 빌드

```bash
npm run build
npm run preview
```

`dist` 폴더를 정적 호스팅하거나 로컬에서 `preview`로 확인할 수 있습니다.

## 주요 기능

- **월간 실적**: 해당 월의 모든 주차를 한 화면에서 입력, 월 합계·연간 예산 열 표시
- **연간 예산**: 계정별 예산 입력, 연 누계 실적 대비 차이(금액·%)·집행률
- **기간 통계**: 분기·반기·연별 주별 실적 및 예산 대비
- **계정 관리**: 추가·삭제·변경, 벌크 입력, JSON Import/Export
- **백업**: 전체 데이터 JSON 백업·복원
- **엑셀**: 월간·기간 통계 화면에서 `.xlsx` 다운로드
